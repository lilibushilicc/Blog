## ⚠️ 避坑提醒

### 1. 国内云流量陷阱

> 阿里云/腾讯云**免费仅存空间，外网下载流量计费**，千万别直接外链图片，容易产生高额账单！

### 2. 信用卡预授权

> 国外部分平台绑信用卡仅预授权，不超免费额度**不会扣费**。

### 3. HTTPS 域名要求

> 自带临时域名大多不支持 HTTPS，想要 HTTPS 必须**绑定自有域名**。

### 1. 七牛云 Kodo（JavaScript 示例）

```js
const qiniu = require('qiniu')

// 配置
const accessKey = '你的 AccessKey'
const secretKey = '你的 SecretKey'
const bucket = '你的存储空间名'

// 生成上传 Token
const mac = new qiniu.auth.digest.Mac(accessKey, secretKey)
const putPolicy = new qiniu.rs.PutPolicy({ scope: bucket })
const uploadToken = putPolicy.uploadToken(mac)

// 上传文件
const config = new qiniu.conf.Config()
const formUploader = new qiniu.form_up.FormUploader(config)
const putExtra = new qiniu.form_up.PutExtra()
const localFile = './test.png'

formUploader.putFile(uploadToken, 'test.png', localFile, putExtra, (err, body, info) => {
  if (err) return console.error(err)
  console.log('上传成功:', body.key, body.hash)
})
```

### 2. Cloudflare R2（S3 兼容 — JavaScript）

```js
// npm i @aws-sdk/client-s3
import { S3Client, PutObjectCommand } from '@aws-sdk/client-s3'

const client = new S3Client({

  credentials: {
    accessKeyId: '你的 R2 Access Key',
```

### 5. MinIO 自建（JavaScript）

```js
import { S3Client, PutObjectCommand } from '@aws-sdk/client-s3'

const client = new S3Client({
  region: 'us-east-1',
  endpoint: 'http://localhost:9000',
  credentials: {
    accessKeyId: 'minioadmin',
    secretAccessKey: 'minioadmin',
  },
  forcePathStyle: true, // MinIO 需要
})

await client.send(new PutObjectCommand({
  Bucket: 'my-bucket',

  Body: require('fs').createReadStream('./test.png'),
}))
```

### 6. Java（Spring Boot + AWS SDK）

```xml
<!-- pom.xml -->
<dependency>
  <groupId>software.amazon.awssdk</groupId>
  <artifactId>s3</artifactId>
  <version>2.29.0</version>
</dependency>
```

```java
import software.amazon.awssdk.auth.credentials.AwsBasicCredentials;
import software.amazon.awssdk.auth.credentials.StaticCredentialsProvider;
import software.amazon.awssdk.regions.Region;
import software.amazon.awssdk.services.s3.S3Client;
import software.amazon.awssdk.services.s3.model.PutObjectRequest;
import java.net.URI;
import java.nio.file.Paths;

// 以 Cloudflare R2 为例（S3 兼容）
S3Client client = S3Client.builder()
  .region(Region.of("auto"))
  .endpointOverride(URI.create("https://<ACCOUNT_ID>.r2.cloudflarestorage.com"))
  .credentialsProvider(StaticCredentialsProvider.create(
    AwsBasicCredentials.create("你的AccessKey", "你的SecretKey")
  ))
  .forcePathStyle(true)
  .build();

client.putObject(
  PutObjectRequest.builder()
    .bucket("你的存储桶名")
    .key("test.png")
    .build(),
  Paths.get("./test.png")
);
System.out.println("上传成功");
```

> ⚠️ **MinIO、Tigris、Backblaze B2** 只需替换 `endpointOverride` 和 `region` 即可复用同一段代码。
