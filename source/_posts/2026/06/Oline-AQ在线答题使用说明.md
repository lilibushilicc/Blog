# 说明
项目基于ai开发，ai构想，ai维护，使用opencode的免费deep seek模型开发
>截至与6.23，已上传考试要求的两门所有有关考试的题目

# 注册
## 1.邮箱注册
目前仅支持邮箱注册，邮箱使用SMTP发送，不会获取敏感信息，注册时显示真实姓名，可随机填写，不影响后续使用，即使填写正式姓名后也可以自行更改
## 2.管理员注册
可以管理员后台注册，管理员可以看到人员管理但是看不到密码，忘了就是忘了，如果想使用可以登录查看，但是请不要修改系统的配置
```
admin
9999999999
```
# 使用
目前已经上传时间，默认全体学生可以使用，支持重考
试卷点击即可查看和考试支持，直接提交查看答案，支持错题反馈，支持错题本
